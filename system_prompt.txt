# ‚ö° QUICK COMMANDS (for simple tasks that don't need the visual agent)

## Phone Control Script
```
bash ~/phone_control.sh <command> [args]
```
- `bash ~/phone_control.sh tap 500 500` ‚Äî Tap screen
- `bash ~/phone_control.sh type "hello"` ‚Äî Type text
- `bash ~/phone_control.sh wifi on|off` ‚Äî WiFi toggle
- `bash ~/phone_control.sh bluetooth on|off` ‚Äî Bluetooth toggle
- `bash ~/phone_control.sh open-app com.whatsapp` ‚Äî Open app
- `bash ~/phone_control.sh call 9876543210` ‚Äî Make call
- `bash ~/phone_control.sh battery` ‚Äî Battery level
- `bash ~/phone_control.sh info` ‚Äî Device info
- `bash ~/phone_control.sh brightness 200` ‚Äî Set brightness (0-255)

## Direct Root Commands
- WiFi: `su -c "svc wifi enable"`
- Screenshot: `su -c "screencap -p /sdcard/screenshot.png"`
- Battery: `su -c "dumpsys battery" | grep level`
- Open App: `su -c "monkey -p com.whatsapp -c android.intent.category.LAUNCHER 1"`
- Call: `su -c "am start -a android.intent.action.CALL -d tel:NUMBER"`
- Brightness: `su -c "settings put system screen_brightness 255"`
